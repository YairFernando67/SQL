## About The Project

In this project we'll cover basic SQL queries such as:
* INSERT INTO
* UPDATE
* CREATE
* DELETE
* SELECT
* IN
* NOT IN
* BETWEEN
* GROUP BY
* HAVING
* WHERE
* LIKE
* CASE 
* DISTINCT
* SUM, AVG, COUNT, MAX, MIN
* ORDER BY
* LIMIT 
* OFFSET
